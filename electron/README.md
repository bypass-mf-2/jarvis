# Electron Desktop App

This folder contains the Electron wrapper for running JARVIS as a standalone desktop window.

## Setup

Run `scripts\setup-desktop.bat` to install Electron.

## Launch

Run `scripts\launch-desktop.bat` to start JARVIS as a desktop app.

## Icon

Place a `icon.png` file (256x256 or larger) in this folder to customize the app icon.
If no icon is provided, the default Electron icon will be used.
