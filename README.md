- **Database:** MySQL
- **AI:** Ollama (LLaMA, CodeLLaMA, LLaVA)
- **Vector DB:** ChromaDB
- **Training:** PyTorch + Transformers + LoRA

## Quick Start
```bash
# Install dependencies
pnpm install

# Setup database
pnpm drizzle-kit migrate

# Pull AI models
ollama pull llama3.2
ollama pull codellama:7b-instruct
ollama pull llava

# Start
pnpm dev
```
